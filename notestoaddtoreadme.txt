Added in the resample_nii_shapes to make a consistent sizing for the segmentator
To run this file:
    A either rename the input and output directories or add in new folders with
        default names "input_dir" "output_dir"
    Move your .nii.gz file into the input directory and run the resample_nii_shapes.py file
    Now use the segmentator tool and input the path as the output directory file
